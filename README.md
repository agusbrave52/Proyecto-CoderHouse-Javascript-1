Proyecto desarrollado para el curso de JavaScript en CoderHouse.

⚠️ Proyecto en proceso — se irán agregando funcionalidades a medida que avance el curso.


📋 Descripción
Simulador de cajero automático que permite a múltiples usuarios autenticarse con su DNI y contraseña para realizar operaciones bancarias básicas como consultar saldo, retirar dinero, ingresar dinero y realizar transferencias entre usuarios.

✨ Funcionalidades

🔐 Inicio de sesión con DNI y contraseña
💰 Consulta de saldo
💸 Retiro de dinero
💵 Ingreso de dinero
🔄 Transferencias entre usuarios registrados
🔙 Navegación entre pantallas con botón "Volver"

🚀 Cómo usar

Cloná el repositorio:

bash   git clone https://github.com/agusbrave52/Proyecto-CoderHouse-Javascript-1.git

Abrí el archivo index.html en tu navegador.
Hacé clic en "Entrar al Cajero".
Ingresá un DNI y contraseña de la tabla de usuarios de prueba.
¡Listo! Ya podés usar el cajero.


Para reiniciar la sesión, recargá la página.
